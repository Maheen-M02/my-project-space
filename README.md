# Portfolio Website

React + TypeScript portfolio project built with Vite, Tailwind CSS, and shadcn-ui components.

## Run locally

1. Install dependencies:
	- `npm install`
2. Start dev server:
	- `npm run dev`

## Build

- Production build: `npm run build`
- Preview build: `npm run preview`

## Test

- Run tests once: `npm run test`
- Watch tests: `npm run test:watch`

## Tech stack

- Vite
- React
- TypeScript
- Tailwind CSS
- shadcn-ui
